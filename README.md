## Features
- Scrapes and displays information about the Aeon group of companies from the Aeon website
- Provides a search interface to find specific companies within the group

## Usage
1. Run the `scrape.js` script to fetch the latest company data and save it to `aeon-company.csv`.
2. Open the `aeon-group2.html` file in a web browser to access the search interface.

## Data / API
The project fetches data from the Aeon company website at https://www.aeon.info/company/group/.
