# aeon

イオングループ企業の一覧を取得・表示するウェブアプリです。イオンの公開情報からグループ企業の名称、株式公開状況、URLなどを収集し、検索機能付きで表示します。

## デモ
https://code4fukui.github.io/aeon/aeon-group2.html

## 機能
- イオングループ企業の一覧表示
- 企業名やURLによる検索機能

## 必要環境
特に必要ありません。ウェブブラウザで閲覧可能です。

## 使い方
ウェブブラウザで上記のデモページを開くと、イオングループ企業の一覧が表示されます。検索ボックスに企業名やURLの一部を入力すると、該当する企業が絞り込まれて表示されます。

## データ
イオン株式会社の公開情報 (https://www.aeon.info/company/group/) から取得したデータを使用しています。
